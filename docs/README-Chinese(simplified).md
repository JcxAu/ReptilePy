## 安装
  - 在 Python38 或 Python 310 的环境下安装本爬虫。
  - 您应该首先安装 request、pillow 和 bs4 库。

## 使用
  - main.py 是本爬虫的主入口点。
  - 支持简体中文和英语。
  - 初始界面为简体中文。
  - 可以改变此程序的语言。只需要找到「语言」选项。
